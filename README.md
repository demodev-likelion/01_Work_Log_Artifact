# 01_Work_Log_Artifact

Claude Desktop / Artifacts 강의 Chapter 1 결과물입니다. Claude의 아티팩트 기능으로 만든 업무 기록 웹페이지 예시와 추가 아티팩트 샘플을 보관하는 자료형 레포입니다.

## 프로젝트 개요

- 목적: Claude 채팅과 아티팩트 기능을 활용해 코딩 없이 간단한 업무 기록 웹페이지를 만들어 보는 실습
- 학습 포인트: Claude 기본 기능, 프롬프트 작성, 아티팩트 생성, HTML 결과물 확인
- 원본 자료 위치: Chapter 1 강의자료

## 포함 파일

```text
artifacts/work_log_tracker.html
artifacts/temporal-sediment.html
```

## 주요 결과물

- `work_log_tracker.html`: 하루 업무를 시간대별로 기록하는 HTML 아티팩트 예시
- `temporal-sediment.html`: Skills/Connector 맛보기에서 사용한 추가 아티팩트 예시

## 실행 방법

별도 설치는 필요하지 않습니다.

```bash
open artifacts/work_log_tracker.html
open artifacts/temporal-sediment.html
```

브라우저에서 HTML 파일을 열어 결과물을 확인합니다.

## 강의 활용 포인트

이 레포는 Chapter 1에서 Claude가 생성한 아티팩트 결과물을 따로 확인하거나 배포 전 샘플로 공유할 때 사용합니다.
